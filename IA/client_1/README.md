# Consignes d'exécution
## 1. Organisation des fichier
Mettre tous les fichiers dans le même dossier :
- Data_Arbres_Clean.csv
- entrainement.ipynb
- script.py
- README.md

## 2. Installation Dépendances
Installer les dépendances listées dans le fichier requirements.txt en exécutant la fonction dans le terminal :
```bash
python3 -m pip install -r requirements.txt
```

## 3. Ordre d'exécution
### Etape 1
Exécuter le fichier entrainement.ipynb qui va générer le fichier arbres_complet_avec_clusters.csv

### Etape 2
Déplacer si besoin le fichier dans votre dossier avec le reste des fichier du point 1

### Etape 3
Exécuter le fichier script.py qui va ouvrir une page internet avec une carte des arbres
